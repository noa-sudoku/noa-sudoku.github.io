# Noa Sudoku

Noa Sudoku 공식 웹사이트입니다.

Noa Sudoku는 광고 없이 스도쿠에만 집중할 수 있도록 만든 iPhone, iPad용
스도쿠 앱입니다. 배너, 팝업, 복잡한 장식 없이 퍼즐을 풀고 기록을 가볍게
남기는 경험에 맞춰 설계했습니다.

## 앱 소개

- 광고 없는 플레이
- 쉬움부터 마스터까지 난이도별 퍼즐 제공
- 날짜별 기록과 난이도별 최고 기록 확인
- 필요할 때만 사용하는 힌트와 되돌리기
- 홈 화면 위젯과 로컬 알림 지원
- iCloud를 사용할 수 있는 환경에서 일부 기록 동기화 지원

## About
