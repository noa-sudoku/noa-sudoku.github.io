# noa-sudoku.github.io

Official website for Noa Sudoku. 광고 없는 스도쿠 앱 공식 홈페이지.

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

The site is built as a Vite multi-page app so `/`, `/features/`, `/support/`,
and `/privacy/` can be used as direct GitHub Pages URLs.
